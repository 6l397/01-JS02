1. Що таке window.document?
window.document представляє собою об'єкт, який відповідає за документ HTML (або XML), що відображається в поточному вікні браузера. Він надає доступ до структури та елементів цього документа, які можна змінювати або взаємодіяти з ними через JavaScript.

2. Яке призначення методу querySelector?
Метод querySelector призначений для пошуку першого елемента в документі, який відповідає заданому CSS селектору. Він повертає перший знайдений елемент або null, якщо елемент не знайдено.

3. Пояснення призначення та відмінності методів textContent та innerHTML.
textContent: Цей метод встановлює або повертає текстовий вміст елемента. Він повертає лише текстовий вміст без HTML-розмітки.
innerHTML: Цей метод дозволяє встановлювати або отримувати HTML-розмітку вмісту елемента. Він повертає HTML-розмітку, яка міститься всередині елемента, включаючи всі його дочірні елементи.

4. Яке призначення методу getElementById?
Метод getElementById призначений для отримання посилання на елемент з документа за його унікальним ідентифікатором (id). Цей метод швидко знаходить єдиний елемент з вказаним id.

5. Поясніть синтаксис запису getElementsByTagName("h1")[0].
getElementsByTagName("h1") вибирає всі елементи <h1> з документа і повертає їх у вигляді колекції (подібно масиву). [0] вибирає перший елемент з цієї колекції (за індексом 0).

6. Які інші події «кліка», які викликаються браузером, коли ми клікаємо по об’єкту HTML-сторінки мишею?
Окрім події click, браузер також може викликати такі події під час кліку:
mousedown: Сталася подія, коли користувач натиснув кнопку миші.
mouseup: Сталася подія, коли користувач відпустив кнопку миші після натискання.
dblclick: Сталася подія подвійного кліку мишею.

7. Яке призначення методу .remove() та element.removeChild(child)?
Метод .remove(): Видаляє елемент з документа, на якому він викликаний.
element.removeChild(child): Видаляє заданий дочірній елемент child з батьківського елемента element.
